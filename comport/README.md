## Русский

### Введение

Приложение доступно по ссылке - https://rand0mRU.github.io/comport

Монитор и плоттер для COM порта (последовательного порта), написанный на HTML и JS для работы в браузере.

### Установка

Проверьте, что у вас установлен npm (https://www.npmjs.com/)

Для установки всех зависимостей выполните

``` bash
npm install
```

Для обновления output.css выполните

``` bash
npm run dev
```
или
``` bash
npm run build
```

### Используете Firefox?

В настоящее время в браузерах, основанных не на Chromium, Web Serial API отсутствует

Попробуйте установить стороннее API, выполняющее работу Web Serial API - https://github.com/kuba2k2/firefox-webserial

## English

### Introduction

The application is available at the link - https://rand0mRU.github.io/comport

Monitor and plotter for COM port (serial port), written in HTML and JS for browser operation.

### Installation

Make sure you have npm installed (https://www.npmjs.com/)

To install all dependencies run

``` bash
npm install
```

To update output.css run

``` bash
npm run dev
```
or
``` bash
npm run build
```

### Are you using Firefox?

Currently, there is no Web Serial API in non-Chromium-based browsers.

Try installing a third-party API that does the job of the Web Serial API - https://github.com/kuba2k2/firefox-webserial
